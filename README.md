## üìà **Key Metrics**

| Metric                         | Formula                                 | Description            |
| ------------------------------ | --------------------------------------- | ---------------------- |
| **CTR (Click-Through Rate)**   | `(Clicks / Impressions) * 100`          | Engagement efficiency  |
| **CPC (Cost per Click)**       | `Spend / Clicks`                        | Cost efficiency        |
| **Conversion Rate**            | `(Approved Conversions / Clicks) * 100` | Ad effectiveness       |
| **ROI (Return on Investment)** | `((Revenue - Spend) / Spend) * 100`     | Campaign profitability |
